# Livestock Analytics

Performance benchmarking and financial analytics for livestock operations

## Business Value

Data-driven decision making, continuous improvement (benchmark-driven), financial transparency, predictive insights

## Key Services

enterprise-profitability, cost-of-production, benchmarking, predictive-models, scenario-planning, dashboards

## Dependencies

- `afenda-database` - Database access
- `afenda-canon` - Standard types and utilities
- `zod` - Schema validation
- `drizzle-orm` - Type-safe SQL queries
