# Cold Chain

Temperature-controlled storage and logistics with continuous monitoring

## Business Value

Regulatory compliance, waste reduction (3-8% savings), product quality assurance, customer satisfaction

## Key Services

temperature-monitoring, shelf-life-tracking, cold-storage-zones, transit-monitoring, excursion-management, cold-chain-analytics

## Dependencies

- `afenda-database` - Database access
- `afenda-canon` - Standard types and utilities
- `zod` - Schema validation
- `drizzle-orm` - Type-safe SQL queries
