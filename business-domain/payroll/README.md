# @afenda-payroll

Enterprise payroll processing and tax compliance.

## Features

- **

Payroll Processing**: Gross-to-net, deductions, direct deposit

- **Tax Withholding**: Federal, state, local, FICA calculations
- **Year-End Reporting**: W-2 generation, 941 reconciliation
- **Garnishments**: Child support, tax levies, bankruptcy
- **Payroll Analytics**: Labor cost, department distribution, variance

## Architecture

**Layer**: 2 (Domain Services)\
**Dependencies**: canon, database, drizzle-orm, zod
