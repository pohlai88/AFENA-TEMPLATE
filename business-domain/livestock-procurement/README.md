# Livestock Procurement

Live animal purchasing with weight-based pricing and quality grading

## Business Value

Cost predictability, quality consistency, risk management, supplier relationships

## Key Services

live-buying, grading, shrink-mortality, forward-contracts, hedging-integration, procurement-analytics

## Dependencies

- `afenda-database` - Database access
- `afenda-canon` - Standard types and utilities
- `zod` - Schema validation
- `drizzle-orm` - Type-safe SQL queries
