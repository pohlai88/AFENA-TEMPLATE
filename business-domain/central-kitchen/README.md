# Central Kitchen

Commissary kitchen for restaurant chains with multi-location distribution

## Business Value

Labor cost reduction at store level (15-25%), food cost reduction (bulk prep efficiency), product consistency across stores, reduce store-level complexity

## Key Services

prep-production, store-demand, distribution-planning, shelf-life, production-costing, ckops-analytics

## Dependencies

- `afenda-database` - Database access
- `afenda-canon` - Standard types and utilities
- `zod` - Schema validation
- `drizzle-orm` - Type-safe SQL queries
