# Royalty Management

Franchise fee calculation, collection, and compliance

## Business Value

Automated royalty revenue recognition, fraud detection (under-reporting), cash flow predictability, reduced disputes

## Key Services

royalty-calculation, sales-reporting, variance-detection, collections, audit-support, royalty-analytics

## Dependencies

- `afenda-database` - Database access
- `afenda-canon` - Standard types and utilities
- `zod` - Schema validation
- `drizzle-orm` - Type-safe SQL queries
