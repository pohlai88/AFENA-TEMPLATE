# Nutrition Labeling

Automated nutrition facts and allergen label generation

## Business Value

Regulatory compliance (FDA 21 CFR 101), label accuracy and risk reduction, time to market (automated label generation), global market access

## Key Services

nutrition-facts, allergen-labeling, claim-validation, label-versioning, multi-language, labeling-analytics

## Dependencies

- `afenda-database` - Database access
- `afenda-canon` - Standard types and utilities
- `zod` - Schema validation
- `drizzle-orm` - Type-safe SQL queries
