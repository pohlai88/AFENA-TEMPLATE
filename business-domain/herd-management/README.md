# Herd Management

Livestock tracking, breeding, health, and performance monitoring

## Business Value

Breeding program optimization, health cost reduction, regulatory compliance (animal ID), performance improvement

## Key Services

animal-registry, breeding-management, health-records, performance-tracking, movement-tracking, herd-analytics

## Dependencies

- `afenda-database` - Database access
- `afenda-canon` - Standard types and utilities
- `zod` - Schema validation
- `drizzle-orm` - Type-safe SQL queries
