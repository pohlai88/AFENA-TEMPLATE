# Feed Mill

Feed formulation, milling, and delivery to farms/livestock operations

## Business Value

Feed cost optimization (5-10% savings), quality consistency, regulatory compliance (FDA, VFD), supply chain control

## Key Services

formulation, milling-production, quality-testing, medicated-feed, feed-delivery, feed-analytics

## Dependencies

- `afenda-database` - Database access
- `afenda-canon` - Standard types and utilities
- `zod` - Schema validation
- `drizzle-orm` - Type-safe SQL queries
