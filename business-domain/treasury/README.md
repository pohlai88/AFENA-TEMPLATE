# @afenda-treasury

Enterprise treasury and cash management.

## Features

- **Cash Positioning**: Multi-bank balances, cash concentration, sweeps
- **Cash Forecasting**: Rolling forecasts, scenario analysis, variance tracking
- **Banking Operations**: Bank reconciliation, statement import, fee analysis
- **Liquidity Management**: Working capital optimization, cash conversion cycle
- **Treasury Analytics**: Cash flow, days cash on hand, liquidity ratios

## Architecture

**Layer**: 2 (Domain Services)\
**Dependencies**: canon, database, payables, receivables, drizzle-orm, zod
