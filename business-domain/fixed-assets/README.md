# @afenda-fixed-assets

Enterprise fixed asset and depreciation management.

## Features

- **Asset Lifecycle**: Acquisition, tracking, transfers, disposals
- **Depreciation**: Multiple methods (SL, DDB, MACRS, units of production)
- **Asset Maintenance**: Scheduled maintenance, repair tracking
- **Physical Inventory**: Asset tagging, physical counts, audits
- **Asset Analytics**: ROI, utilization, depreciation schedules

## Architecture

**Layer**: 2 (Domain Services)\
**Dependencies**: canon, database, accounting, drizzle-orm, zod
