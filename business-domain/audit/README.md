# @afenda-audit

Comprehensive audit trail, change tracking, and compliance reporting.

## Features

- **Audit Logging**: Immutable audit trail for all entity changes and system access
- **Change Tracking**: Field-level change history with before/after snapshots
- **Compliance Reporting**: SOX, GDPR, and regulatory compliance reports
- **Data Retention**: Automated archival and purging with legal hold support
- **Anomaly Detection**: Suspicious activity detection and alerting

## Architecture

**Layer**: 2 (Domain Services)\
**Dependencies**: canon, database, drizzle-orm, zod
