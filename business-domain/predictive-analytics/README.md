# @afenda-predictive-analytics

Enterprise predictive analytics and ML.

## Services

### ml-models.ts

- `trainModel` - Train ML model
- `evaluateModel` - Evaluate model performance

### forecasting.ts

- `forecastRevenue` - Forecast revenue
- `predictDemand` - Predict product demand

### optimization.ts

- `optimizeInventory` - Optimize inventory levels
- `optimizePricing` - Optimize pricing strategy

### scenario-planning.ts

- `createScenario` - Create what-if scenario
- `compareScenarios` - Compare scenario outcomes

### predictive-insights.ts

- `identifyPatterns` - Identify data patterns
- `detectAnomalies` - Detect anomalous behavior

Layer 2 (Domain Services), pure functions, no logging, Zod validation.
