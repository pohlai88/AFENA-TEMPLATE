# Recipe Management

Food recipe/formula management with version control and costing

## Business Value

Product consistency and quality, regulatory compliance (FDA labeling), cost accuracy, R&D product development

## Key Services

recipe-versioning, ingredient-scaling, nutrition-calculation, allergen-tracking, recipe-costing, substitution-rules

## Dependencies

- `afenda-database` - Database access
- `afenda-canon` - Standard types and utilities
- `zod` - Schema validation
- `drizzle-orm` - Type-safe SQL queries
