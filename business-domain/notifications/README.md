Multi-channel notification delivery and alert management.

## Features

- **Notification Delivery**: Email, SMS, push notifications, in-app messages
- **Template Management**: Reusable templates with variable substitution
- **Subscription Management**: User preferences and opt-in/opt-out
- **Delivery Tracking**: Status tracking, engagement analytics, retry logic
- **Alert Management**: Alert rules, escalation workflows, priority handling

## Architecture

**Layer**: 2 (Domain Services)\
**Dependencies**: canon, database, drizzle-orm, zod
