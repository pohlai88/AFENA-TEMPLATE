# @afenda-warehouse

Enterprise warehouse management system (WMS) operations.

## Features

- **Picking**: Wave planning, batch picking, pick-to-light, pick confirmation
- **Packing**: Packing stations, cartonization, packing slips
- **Cycle Counting**: ABC counting, location audits, variance resolution
- **Slotting**: Optimize bin assignments, velocity-based slotting
- **Warehouse Analytics**: Productivity, accuracy, space utilization

## Architecture

**Layer**: 2 (Domain Services)\
**Dependencies**: canon, database, inventory, sales, drizzle-orm, zod
