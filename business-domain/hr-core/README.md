Employee master data and organizational structure management.

## Features

- **Employee Management**: Hire, transfer, terminate, employee lifecycle
- **Organizational Structure**: Departments, positions, reporting lines, org charts
- **Employee Data**: Personal info, employment history, document management
- **Competency & Skills**: Skill matrices, competency models, gap analysis
- **Workforce Analytics**: Headcount, turnover, span of control, forecasting

## Architecture

**Layer**: 2 (Domain Services)\
**Dependencies**: canon, database, drizzle-orm, zod
